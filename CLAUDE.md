# 部活動地域展開 指導謝金管理アプリ 要件定義

## プロジェクト概要

北海道の自治体教育委員会が運用する、部活動地域展開における地域クラブ指導者への指導謝金を管理するWebアプリケーション。指導者が月報をアプリ上で入力・提出し、事務局が謝金を自動計算・管理できるシステム。

---

## 技術スタック

| レイヤー | 技術 | 理由 |
|---------|------|------|
| フロントエンド | HTML / CSS / JavaScript (Vanilla) | 軽量・無料ホスティング可 |
| ホスティング | GitHub Pages または Vercel（無料枠） | コスト0、即日公開可能 |
| バックエンド | Google Apps Script (GAS) | 無料・Googleスプレッドシート連携が容易 |
| データベース | Google Sheets | 無料・事務局がそのまま閲覧・編集可能 |
| 通知 | GAS + Gmail API | 無料 |
| PDF出力 | jsPDF（ブラウザ側） または GAS HTML出力 | 無料 |

---

## ユーザー区分

### 1. 指導者（最大100名）
- 月報の入力・一時保存・提出
- 自分の提出済み月報の閲覧（修正不可）
- 提出期限のリマインド通知受信（メール）

### 2. 事務局（最大5名）
- 全指導者の月報提出状況の管理
- 謝金自動計算結果の確認・修正
- 指導者マスタの管理
- Googleスプレッドシートへのエクスポート
- 給与明細のPDF出力・印刷
- 月報提出時の通知受信（メール）

---

## 認証・アクセス

- **ログイン機能なし**
- URLを知っていれば誰でもアクセス可能
- 指導者画面URL（例）: `https://example.com/report`
- 事務局画面URL（例）: `https://example.com/admin`
- セキュリティ対策: 事務局URLは推測困難なランダム文字列を含める

## 画面構成

### 指導者画面

#### 1. 月報入力画面 `/report`
**入力項目：**
- 指導者氏名（テキスト入力 または プルダウン選択）
- クラブ名（テキスト入力 または プルダウン選択）
- 対象年月（年・月セレクト）
- 区分（平日 / 休日 / 長期休暇 / 大会引率）

- 開始時刻（HH:MM）
- 終了時刻（HH:MM）
- 指導時間（自動計算・表示のみ）
- 旅費申請
  - 交通手段（バス代 / JR代）
  - 行先（テキスト）
  - 金額（数値）
- 備考（任意）

**機能：**
- 日付ごとに1行入力（最大31行）
- 一時保存ボタン（何度でも修正可能）
- 提出ボタン（押下後は修正不可）
- 提出前に確認ダイアログ表示
- 提出後に完了画面表示

#### 2. 提出確認画面
- 入力内容の最終確認
- 謝金計算結果のプレビュー表示
- 「提出する」「戻って修正する」ボタン

#### 1. ダッシュボード `/admin`

#### 3. 謝金計算・支払明細画面

- 指導者選択・年月選択

## 謝金計算ロジック

### 基本ルール

```
時給:
  メイン指導者: 1,600円/時間
  サブ指導者:   1,100円/時間

源泉徴収率: 10.21%（円未満切捨）
差引支払額 = 謝金総額 - 源泉徴収額
```

### 指導時間の計算

#### 一般指導者
```
指導時間 = 終了時刻 - 開始時刻
```

#### 教員指導者（平日・長期休暇の平日のみ）
```
教員勤務終了時刻 = 16:45
有効指導時間 = MAX(終了時刻 - MAX(開始時刻, 16:45), 0)
```

### 上限・切捨て

```
区分別上限:
  平日:    2時間
  休日:    3時間
  長期休暇: 3時間
  大会引率: 4時間

謝金計算時間 = FLOOR(MIN(有効指導時間, 上限時間) × 4, 1) / 4
※ 15分単位切捨（例: 1時間50分 → 1時間45分）
```

### 謝金額計算

```
謝金額 = ROUND(謝金計算時間 × 時給, 0)
源泉徴収額 = ROUNDDOWN(謝金額 × 0.1021, 0)
差引支払額 = 謝金額 - 源泉徴収額
```

## Google Apps Script (GAS) 構成

### エンドポイント一覧

```
POST /exec?action=saveReport      月報の一時保存
POST /exec?action=submitReport    月報の提出
GET  /exec?action=getReport       月報データの取得
GET  /exec?action=getInstructors  指導者マスタの取得
POST /exec?action=updateMaster    指導者マスタの更新
GET  /exec?action=getDashboard    ダッシュボードデータ取得
POST /exec?action=calcFee         謝金計算の実行
POST /exec?action=updateFee       謝金の手動修正
GET  /exec?action=exportSheet     スプレッドシートエクスポート
```

### トリガー設定

```
時間ベーストリガー:
  毎月1日 00:00  → 翌月5日締切のリマインドメール送信開始
  毎月3日 09:00  → 未提出者へリマインドメール送信
  毎月4日 09:00  → 未提出者へ最終リマインドメール送信
  毎月5日 18:00  → 事務局へ未提出者リスト通知

提出イベントトリガー:
  月報提出時     → 事務局全員にメール通知
```

## 制約・注意事項

- GASの実行制限: 1回の実行は最大6分、1日の実行時間は6時間
- Googleスプレッドシートの同時編集: 競合が起きないようUUIDで行を特定
- 事務局URLは公開リポジトリにコミットしない（環境変数または別ファイル管理）
- 個人情報（口座番号等）はGoogleスプレッドシートのアクセス権限を事務局のみに制限
- 月報データは年度ごとにシートを分けて管理（シート名: `月報データ_2026`）
